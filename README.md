## 🧠 About Wazuh

Wazuh is a **free, open-source security platform** that provides:

- 🔍 Intrusion Detection (HIDS/NIDS)

- 🛡️ File Integrity Monitoring (FIM)
- ⚠️ Vulnerability Detection
- 📋 Security Configuration Assessment (SCA)
- ⚡ Active Response & Incident Handling
- ☁️ Cloud Workload Protection

## 🧩 Custom Detection Rules

The project includes **70+ custom XML rules** (IDs `100200` – `102499`) covering:

- Failed logins & brute‑force detection
- Suspicious process execution & PowerShell abuse
- File integrity monitoring (critical system files)
- Registry persistence (Run/RunOnce keys)
- Windows Defender tampering
- Suspicious DNS TLDs (`.xyz`, `.top`, etc.)
- Security log clearing & remote interactive logons

These rules are ready to be copied into `/var/ossec/etc/rules/local_rules.xml` on the Wazuh manager.
